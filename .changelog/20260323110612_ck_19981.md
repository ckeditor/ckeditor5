Added config-only constructor and `create()` overloads for all editor types (`ClassicEditor`, `InlineEditor`, `BalloonEditor`, `DecoupledEditor`, `MultiRootEditor`). The legacy signatures where the first argument is a source element or data string are now deprecated but remain supported. Introduced `config.attachTo` for `ClassicEditor` and `RootConfig.element` for per-root element binding. Updated `EditorWatchdog` and `ContextWatchdog` to support config-based editor creation without a separate `sourceElementOrData` argument, including adjusted restart behavior and signature detection.
