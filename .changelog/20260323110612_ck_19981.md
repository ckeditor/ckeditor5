All editor types (`ClassicEditor`, `InlineEditor`, `BalloonEditor`, `DecoupledEditor`, `MultiRootEditor`) can now be created using config only, without passing a source element or data string as the first argument. The previous way of passing a source element or data string as the first argument is now deprecated but still works. Added `config.attachTo` for `ClassicEditor` and `RootConfig.element` for per-root element binding. Updated `EditorWatchdog` and `ContextWatchdog` to support config-based editor creation.
