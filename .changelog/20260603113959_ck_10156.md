Fixed an issue where multiple editor instances on the same page could interfere with each other's emoji data when they shared the same `definitionsUrl` but used different `useCustomFont` settings. Depending on which editor initialised first, some editors could silently display an incorrect set of emoji — for example showing a restricted list when a full one was expected, or the other way around.
