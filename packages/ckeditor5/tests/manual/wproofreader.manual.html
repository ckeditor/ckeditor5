Wproofreader

Proofreader requires a valid license to work, otherwise it returns 403. Read on the dev KB article to set it up locally.

Note: the plugin starts checking only after focusing the editor.

Inline typo correction

  1. Focus the editor.
  2. Wait until typos are indicated.
  3. Hover mouse cursor over "witth".
  4. Pick "with" from appearing context menu.

Inline grammar correction

  1. Focus the editor.
  2. Wait until typos are indicated.
  3. Hover mouse cursor over (the first) "However".
  4. Pick "However," from appearing context menu.

Dialog checking

  1. Focus the editor.
  2. Hover a spell check icon in the bottom-right corner of the editable.
  3. Click "Proofread in dialog" option.
  4. Use it to fix a typo.
