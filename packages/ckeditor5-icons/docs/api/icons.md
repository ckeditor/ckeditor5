# CKEditor&nbsp;5 icons

This package contains icons &ndash; for CKEditor&nbsp;5.

## Documentation

If you want to customize the default icons used in the editor, see the {@link getting-started/setup/customizing-icons Customizing icons} guide.

## Installation

This package is part of our open-source aggregate package.

```bash
npm install ckeditor5
```

## Contribute

The source code of this package is available on GitHub in [https://github.com/ckeditor/ckeditor5/tree/master/packages/ckeditor5-icons](https://github.com/ckeditor/ckeditor5/tree/master/packages/ckeditor5-icons).
