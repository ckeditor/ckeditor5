Table Caption

Adding a caption to a table

  1. Put selection in the table.
  2. In the opened toolbar click the last icon on the right to turn on the caption.
  3. The table caption should:
    • show up above the table with a highlight effect
    • be focused
    • have a placeholder Enter table caption visible

Editing the caption

  1. Once there is a caption visible, type in any text.
  2. Turn off the caption.
  3. Toolbar button for toggling the caption should change from Toggle caption off to Toggle caption on.
  4. Turn on the caption.
  5. Toolbar button for toggling the caption should change from Toggle caption on to Toggle caption off.
  6. The text typed previously should be kept in the caption.
