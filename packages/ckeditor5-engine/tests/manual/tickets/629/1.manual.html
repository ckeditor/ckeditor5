Selection infinite loop #629

Before testing open console.

  1. Select part of text.

  2. Click bold or press ctrl + b.

  3. There should be no errors or warnings in console.

  4. Put caret somewhere in text.

  5. Press left arrow, then right arrow.

  6. Repeat as fast as you can for 2-3 seconds.

  7. There should be no errors or warnings in console.
