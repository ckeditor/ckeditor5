Keep in mind that this is just a simple demo.
Highlighted markers do not merge (in model), remove button removes the first marker range which contains selection anchor.
Once highlighted, highlight cannot be overwritten.
Multi-element ranges are not supported
Moving to the start button works only for flat ranges
These are all known limitations of the demo.
