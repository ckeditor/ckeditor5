Source editing

  1. Start the source editing mode and verify that the HTML code for a heading, paragraph, table and both lists is shown.
  2. Verify that all buttons, except the source editing, are disabled and off. The source editing button should be enabled and on.
  3. Make some changes in the HTML code.
  4. Go back from the source editing mode and verify that changes made in the previous step are present in the document content.
  5. Add a new table (or use the existing one) and click/tap in the table cell to open the balloon panel.
  6. While the balloon panel is shown, start the source editing mode and verify that this balloon panel is gone.
  7. Verify if Undo works with this feature.
    1. Do some changes in the document, toggle the source editing on and off. Check if Undo works. Check if no empty undo step has been added.
    2. Toggle the source editing on. Do some changes. Toggle it off. You should be able to undo these changes with the Undo button.
